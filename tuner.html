We use the YIN algorithm to find the pitch of input sound.
The input signal, obtained from the Web Audio API, comes in as a discrete time-series array of amplitudes. We use the following autocorrelation function to quantify the similarity between a signal and itself delayed by some lag τ . r t ( τ ) = ∑ j = t + 1 t + W − τ x j x j + τ Here, r t ( τ ) is the autocorrelation function at the time intex t , with a lag of τ . W is the integration window size, or the time frame in consideration. We subtract τ from the upper summation bound to lessen the effect of larger lag sizes, with the assumption that the fundemental frequency is closer to the origin point t .

The Weiner-Khinchin theorem (in the discrete time case) allows us to compute the autocorrelation funtion using a Fourier Transform.
